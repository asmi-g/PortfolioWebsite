CIFAR10 Image Classification

This project implements and trains deep learning models using TensorFlow and a convolutional neural network (CNN) to achieve an 80% accuracy in image classification using the CIFAR-10 dataset.

I also used the NumPy, Matplotlib, and scikit-learn libraries for data preprocessing, visualization, and model evaluation.

  • GitHub Repo

Diabetes Prediction w/ FFN

First, I performed data preprocessing and visualization on the diabetes dataset using pandas and Matplotlib.

I then used a Feed-Forward Neural Network (FFNN) to analyze diabetes-related health indicators, using TensorFlow for model development and training to predict the likelihood of developing diabetes given the aforementioned indicators.

  • GitHub Repo

Amazon Reviews Sentiment Analysis

To analyse the sentiment of a dataset of amazon reviews, I used several techniques.

The first was the Natural Language Toolkit (NLTK) library to explore and assess the VADER model. I also used the Roberta pre-trained model to do the same, as well as a pipeline from Hugging Face transformers. After the analysis, I compared the accuracy of these.

I also visualized the data using Seaborn.

  • GitHub Repo

GitHub Web Crawler

For this project, I used BeautifulSoup to extract the html data of the GitHub popular topics webpage, as well as individual topic webpages into BeautfulSoup objects.

I then identified different HTML attributes to insert that data into Pandas dataframes, and automated that data extraction to CSV files for an organized way to store and record the crawled data.

  • GitHub Repo

COVID19 Data Exploration & Visualization

This project consisted of conducting SQL queries on a COVID19 global dataset, and exporting the resulting tables into Tableau.

I then created graphs and visual representations from that data, and created a clean and comprehensive dashboard to communicate those insights.

USA Traffic Analysis

This project uses a Kaggle dataset for historical data on US, country-wide traffic.

I first prepared and cleaned the data using pandas dataframes, then used seaborn to visualize those very dataframes.

Finally, I created a heatmap using the folium library, and expanded upon important references from the analyzed data.
